6. App Icon

The favicon features the ΔS<0 thermodynamic symbol — a reference to Rudolf Clausius and entropy. Adapts to the active OS/app theme.

Dark mode: #0d1117 background, #E95378 glyph
Light mode: #ffffff background, #2BA298 glyph
Corner radius: 28px (on 128×128 viewBox)
Icon font: Helvetica Neue, weight 800/700
Detection: prefers-color-scheme media query + JS data-theme

8. Layout & Grid

The app uses a shell layout with a fixed topbar, collapsible sidebar, and a responsive content grid.

Topbar: 40px height, position: fixed, frosted-glass var(--nav-bg) with backdrop-filter
Sidebar: resizable (120–600px), default 200px, width stored in localStorage
Content grid: repeat(auto-fill, minmax(680px, 1fr)), gap 14px
Main padding: 14px 16px 40px
Modal max-width: ~640px (run/stats), ~740px (settings), ~580px (advisor)

Responsive breakpoints:

10. Theme Switching

Theme is controlled by a data-theme attribute on <html>, persisted in localStorage.

Storage key: localStorage['clausius.theme']
Values: "system" | "light" | "dark"
System detection: matchMedia('(prefers-color-scheme: dark)')
Flash prevention: Inline <script> in <head> sets data-theme before first paint
Favicon: Dynamically generated SVG data URI — rose-pink on dark (#0d1117), teal on light (#ffffff)
CSS scope: :root for light defaults, [data-theme="dark"] for dark overrides
